# REST Assured API Automation

## What this project does
End-to-end API test for Place Management API

## Test Flow
1. POST - Add a new place, extract place_id
2. PUT  - Update address using place_id
3. GET  - Verify address was updated successfully

## Tech Used
- Java
- REST Assured 6.0
- TestNG
- Maven

## How to Run
1. Clone this repository
2. Open in Eclipse as a Maven project
3. Run PlaceApiTest.java as Java Application
